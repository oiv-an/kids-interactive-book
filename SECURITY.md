# Security Policy

## Reporting a Vulnerability
Если вы нашли уязвимость, пожалуйста **не создавайте публичный issue**.

Отправьте детали на email владельца репозитория (будет добавлен после публикации) или используйте GitHub Security Advisories (если включено).

## Scope
Проект детский:
- никаких сторонних трекеров/аналитики
- никаких аккаунтов/PII
- контент должен быть безопасным

Спасибо за ответственное раскрытие.
