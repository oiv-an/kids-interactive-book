# docs/

Эта папка для **открытого процесса разработки** (Build in Public).

## Что сюда класть
- Промпты для AI (история диалогов, удачные формулировки, версии промптов)
- Черновики сценариев/контента (без персональных данных)
- Заметки по UX/контенту для детей
- Технические решения и их причины (коротко)

## Рекомендуемая структура
- `docs/prompts/` — промпты по датам или по фичам
- `docs/decisions/` — короткие ADR (Architecture Decision Records)
- `docs/content/` — контент/тексты/списки объектов (если нужно)

## Правила
- Не добавлять секреты (токены, ключи, приватные ссылки)
- Не добавлять персональные данные детей/родителей
- Если нужно хранить большой бинарник — лучше внешнее хранилище и ссылка

## Стартовый шаблон промпта
Создай файл в `docs/prompts/` вида: `YYYY-MM-DD_topic.md` и пиши:

```md
# Контекст
- Что делаем
- Текущее состояние

# Цель
- Какой результат нужен

# Ограничения
- Браузеры, оффлайн, i18n, и т.д.

# Промпт
...

# Результат
- Что получилось
- Что улучшить
